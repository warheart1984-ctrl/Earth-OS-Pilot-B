# EarthOS → CIEMS Constitutional Alignment

## Purpose

Map EarthOS artifacts to CIEMS sovereignty layers, treat EarthOS pilots as CIEMS substrations, and use CIEMS promotion rules to graduate pilots from experimental to ratified arenas.

## Layer Mapping

| CIEMS Layer | EarthOS Artifact |
|-------------|-----------------|
| Constitution | CESF v1.1 Core Principles (determinism, non-compensation, append-only, replay equivalence, transparency) |
| Specification | CESF v1.1 Normative Standards (CAS, CMS, CCT, CPBA, CPRM) |
| Schema | CAL, CRC, CPBA, CPRM JSON Schemas |
| Conformance | CCT Ω∞ L0–L5 test suites |
| Implementation | CGE reference (EarthOS Pilot A), Federated CGE (Pilot B), Robotics CGE (Pilot C) |
| Deployment | EarthOS Pilot A sandbox, Pilot B federated network, Pilot C robotics simulation |
| Stewardship | Evidence Stewardship Board, Promotion Authority, Constitutional Review Council, Ratification Assembly |

## Substrate → Substration → Promotion Model

EarthOS pilots are **substrations** within the CIEMS sovereignty stack:

| Substration | Arena | Promotion Path |
|-------------|-------|---------------|
| Pilot A | Single-node constitutional governance | L3 conformance → CPBA clearance → CPRM R5 → Ratification |
| Pilot B | Multi-cluster federated governance | L5 conformance + Pilot A ratification → Federation review → Federated ratification |
| Pilot C | Constitutional robotics | L5 conformance + Safety envelope validation → Robotics review → Robotics ratification |

## Graduation Rules

A pilot graduates from **experimental** to **ratified arena** when:

1. All applicable CCT levels pass (100%)
2. CPBA: PROMOTION_ALLOWED (all barriers satisfied)
3. CPRM: R5 readiness (all 10 contracts pass)
4. Governance Review: majority approval
5. Ratification Assembly: supermajority vote (≥2/3)

## CIEMS-Enhanced Governance

| Governance Body | CIEMS Role |

| Promotion Authority | Evaluates promotion readiness across all 6 CIEMS layers |
| Constitutional Review Council | Ensures constitutional consistency across substrations |
| Ratification Assembly | Ratifies substrations as sovereign arenas |
