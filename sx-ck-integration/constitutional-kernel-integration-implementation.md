## Usage Examples

### Integrating Constitutional Kernel

```typescript
import { ConstitutionalKernelIntegration } from './sx-ck-integration/constitutional-kernel-integration';

const kernelIntegration = new ConstitutionalKernelIntegration(kernel);
const federationConfig = {
  domains: ['cluster-a', 'cluster-b'],
  treatyType: 'SAFETY',
  constitutionalBasis: 'Article 6: Multi-World Federation'
};

const integrationResult = await kernelIntegration.integrateConstitutionalKernel(federationConfig);
console.log(`Integration Status: ${integrationResult.status}`);
console.log(`Constitutional Compliance: ${integrationResult.compliance}`);
```

### Validating Federation Constitutional Compliance

```typescript
const validationResult = await kernelIntegration.validateFederationConstitutional({
  treatyId: 'treaty-123',
  domains: ['cluster-a', 'cluster-b'],
  operations: ['policy-sync', 'evidence-exchange']
});
console.log(`Constitutional Compliance: ${validationResult.compliance}`);
console.log(`Authority Graph: ${validationResult.authorityGraph}`);
```
